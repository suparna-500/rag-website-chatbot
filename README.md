# RAG Website Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that answers questions based on the content of a website.

## Features Completed

- Project setup
- Recursive website scraper
- Same-domain crawling
- HTML content extraction
- Structured text extraction (headings, lists, tables)
- Clean text preprocessing

## Upcoming Features

- Text chunking
- Embedding generation
- ChromaDB vector database
- RAG retrieval pipeline
- Streamlit chat interface
